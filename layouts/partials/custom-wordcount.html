{{/* 自定义字数统计 - 支持中英文混合 */}} {{- $content := .Content -}} {{- /* 移除HTML标签 */ -}} {{- $textContent := replaceRE "<[^>]+>" "" $content -}} {{- /* 统计中文字符数 - 使用Go的Unicode正则表达式格式 */ -}} {{- $chineseChars := findRE "[\\x{4e00}-\\x{9fa5}]" $textContent -}} {{- $chineseCount := len $chineseChars -}} {{- /* 统计英文字母数 */ -}} {{- $englishChars := findRE "[a-zA-Z]" $textContent -}} {{- $englishCount := len $englishChars -}} {{- /* 统计数字 */ -}} {{- $numbers := findRE "[0-9]" $textContent -}} {{- $numberCount := len $numbers -}} {{- /* 总字数 = 中文字符 + 英文字符 + 数字 */ -}} {{- $total := add $chineseCount $englishCount $numberCount -}} {{- return $total -}}
