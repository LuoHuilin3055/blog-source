**写一份指南，避免下次提交作业时又去骚扰AI**

# 1.确保当前工作区干净
```bash
git status
```

# 2.创建并切换到新分支
```bash
git checkout -b 0x02
```

# 3.添加文章文件
- 需要将文章放在本地目录中
```bash
git add article.md
```

# 4.提交更改
```bash
git commit -m "添加文章"
```

# 5.推送新分支到远程仓库
```bash
git push -u origin 0x02
```
