# ⚡️ 最后复习：Python考试速战速决版 ⚡️

## 一、**基本概念与写法**（必背）

### 1️⃣ **输出和输入**

- 输出：`print("内容")`
    
- 输入：`变量 = input("提示信息")`
    

```python
print("请输入数字：")
num = input()
print("你输入的是：", num)
```

### 2️⃣ **变量**

变量是存储数据的地方：

```python
a = 5  # 变量a存储5
```

---

## 二、**条件判断（if语句）**（常见题）

### **基本结构**

```python
if 条件:
    执行的代码
else:
    另一种情况执行的代码
```

### **示例：判断成绩**

```python
score = int(input("请输入成绩："))
if score >= 60:
    print("及格")
else:
    print("不及格")
```

---

## 三、**循环（for语句）**（必考，简单）

### **基本结构：**

```python
for 变量 in 范围：
    执行的代码
```

### **例子：打印数字**

```python
for i in range(5):  # 打印 0, 1, 2, 3, 4
    print(i)
```

---

## 四、**列表操作（常考）**

### **列表**存储一堆数据：

```python
areas = [100, 200, 150, 300]
```

### **常用操作：**

- 求和：`sum(areas)`
    
- 最大值：`max(areas)`
    
- 个数：`len(areas)`
    

### **例子：求平均值**

```python
total = sum(areas)
avg = total / len(areas)
print("平均值：", avg)
```

---

## 五、**条件筛选（最常考）**

### **筛选列表中的数据：**

```python
for a in areas:
    if a > 150:
        print(a)  # 输出大于150的值
```

## 六、**字典（数据存储）**（有时候考）

## 七、**文件读取**（如果涉及，基本这么写）

```python
file = open("data.txt", "r")  # 打开文件
for line in file:
    print(line)  # 输出每一行
file.close()  # 关闭文件
```

## 八、**考试常见题型（直接套用）**

### **1. 计算平均值**

```python
areas = [100, 200, 150, 80]
total = sum(areas)
avg = total / len(areas)
print("平均值：", avg)
```

### **2. 计算最大值**

```python
print(max(areas))  # 输出最大值
```

### **3. 筛选大于某个值的数据**

```python
for a in areas:
    if a > 150:
        print(a)  # 输出大于150的地块
```

### **4. 条件统计数量**

```python
count = 0
for a in areas:
    if a > 150:
        count += 1
print(count)  # 输出大于150的地块数量
```

## 九、**必背模板（考试中用）**

### **全能模板：**

```python
areas = [100, 200, 150, 80]

# 计算总和和平均值
total = sum(areas)
avg = total / len(areas)

# 输出大于平均值的地块
for a in areas:
    if a > avg:
        print(a)

# 输出平均值
print("平均值：", avg)
```
