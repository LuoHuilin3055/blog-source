# 网络空间安全概述

- 网络空间安全概念演变
```plain
  阶段1: 通信保密 ──→ 阶段2: 计算机安全 ──→ 阶段3: 信息系统安全 ──→ 阶段4: 网络空间安全
(古典密码)          (1970s COMPUSEC)     (1990s INFOSEC)     (21世纪 第五大空间)
    │                   │                        │                       │
    ▼                   ▼                        ▼                       ▼
  数据加密            DES/TCSEC                互联网安全            全方位国家安全
```
  - 网络空间安全基本属性
	  - | 攻击形式  | 对应安全属性     | 说明        |
| ----- | ---------- | --------- |
| 偷看/窃听 | **机密性(C)** | 防止信息泄露    |
| 篡改/删节 | **完整性(I)** | 防止数据被修改   |
| 拦截/中断 | **可用性(A)** | 确保系统正常运行  |
| 重放攻击  | **真实性**    | 防止信息被重复利用 |
| 伪装/冒充 | **实体真实性**  | 身份鉴别      |
| 否认发送  | **不可否认性**  | 防止抵赖行为    |
- 安全服务与安全机制（ISO 7498-2)
	- | 安全服务         | 核心功能          |
| ------------ | ------------- |
| **鉴别**       | 对等实体鉴别、数据原发鉴别 |
| **访问控制**     | 控制实体对资源的访问权限  |
| **数据完整性**    | 防止信息遭受篡改等破坏   |
| **数据保密性**    | 使数据不被非授权泄露    |
| **抗抵赖/不可否认** | 原发抗抵赖、接收抗抵赖   |

# 传统密码
- 什么是密码学
	- 密码学是研究密码系统或通信安全的一门学科，分为密码编码学和密码分析学
	- **密码编码学**：研究如何对信息编码以实现信息和通信安全
	- **密码分析学/密码破译学**：研究如何破译或攻击受保护信息
- ## 传统密码
	- 主要特点：数据的安全基于算法的保密
	- |分类层级|名称|包含的具体算法/子类|文档说明|
|:--|:--|:--|:--|
|一级分类|传统密码|1. 代换密码  <br>2. 置换密码|传统密码主要分为这两大类。|
|二级分类|代换密码  <br>(Substitution)|单表代换：  <br>• 棋盘密码  <br>• 凯撒密码 (Caesar)  <br>• 移位密码  <br>• 替换密码 (Simple Substitution)  <br>  <br>多表代换：  <br>• Vigenère (维吉尼亚) 密码  <br>  <br>多字母代换：  <br>• Hill (希尔) 密码|明文中的字符被替换成密文中的另一个字符。|
|二级分类|置换密码  <br>(Permutation)|• 滚筒密码 (Scytale)  <br>• 栅栏技术 (Rail-fence)|又称换位密码。没有改变明文字母，只改变了字母的出现顺序。|
- ```text
传统密码 (Traditional Cipher)
├── 代换密码 (Substitution) - [字符被替换]
│   ├── 单表代换 (Monoalphabetic)
│   │   ├── 棋盘密码
│   │   ├── 移位密码 (含凯撒密码)
│   │   └── 替换密码
│   ├── 多表代换 (Polyalphabetic)
│   │   └── Vigenère 密码
│   └── 多字母代换 (Polygraphic)
│       └── Hill 密码
│
└── 置换密码 (Permutation) - [顺序被改变]
    ├── 滚筒密码 (Scytale)
    └── 栅栏技术 (Rail-fence)
  ```
